## 🚀 快速开始

### 场景一：使用天店模板导入商品

```bash
# 1. 访问界面
http://localhost:8001

# 2. 上传文件
拖拽或点击上传商品数据文件

# 3. 选择模板
选择"思迅天店（模板文件）"

# 4. 填写编码
填写供应商编码和商品类别

# 5. 导出文件
点击"导出 Excel"下载

# 6. 导入系统
登录思迅天店系统导入
```

详细步骤：[天店模板使用说明.md](./天店模板使用说明.md)

---

### 场景二：开发自定义模板

```bash
# 1. 准备模板文件
创建 Excel 文件，第一行写字段名

# 2. 放置模板
将文件放到 moban/ 目录

# 3. 更新前端
在 index.html 中添加选项

# 4. 测试导出
上传测试文件验证
```

详细步骤：[ERP_TEMPLATE_GUIDE.md](./ERP_TEMPLATE_GUIDE.md)

## 🔧 配置文件位置

### 后端配置

```
core/exporter/erp_templates/__init__.py
├── ERP_TEMPLATES 字典
│   ├── default (默认模板)
│   ├── sixun_tiandian (思迅天店配置)
│   └── sixun_shangyun_x (思迅商云X配置)
└── get_template() 函数
```

### 前端配置

```
templates/index.html
└── ERP模板下拉框
    ├── 默认模板
    ├── 思迅天店（自动映射）
    ├── 思迅天店（模板文件） ← 新增
    └── 思迅商云X（商品档案）
```

### 模板文件

```
moban/
├── 天店商品导入模板.xlsx ← 你的模板
└── 商品档案模版.xls
```

## 💡 使用建议

### 推荐配置

| 使用场景 | 推荐模板 | 理由 |
|---------|---------|------|
| 思迅天店标准导入 | **模板文件方式** | 格式完整、样式保留 |
| 自定义字段需求 | 配置模板方式 | 灵活可控 |
| 数据快速查看 | 默认模板 | 简单直接 |

### 最佳实践

1. **优先使用模板文件方式**
   - 格式和样式完整保留
   - 完全符合系统要求
   - 维护简单（替换文件即可）

2. **手工编码统一管理**
   - 建立供应商编码对照表
   - 建立商品类别对照表
   - 导入时批量填写

3. **数据验证**
   - 导出前检查必填字段
   - 验证条码、价格格式
   - 小批量测试后再大批量导入
